data encoding
ㄴ ML 알고리즘은 숫자만 받아들임
   문자 데이터를 숫자 데이터로 변경해야함
1) label encoding : 문자 -> 숫자
  상품분류       가격            상품분류       가격
    TV           1000000           0            1000000
    냉장고       1500000           1            1500000
    세탁기        500000   ->      2             500000
    컴퓨터       2000000           3            2000000

2) One-Hot encoding : 
    ㄴ feature 값의 유형에 따라서 새로운 feature를
       추가해서 고유값에 해당하는 칼럼만 1을 표시하고
       나머지는 0을 표시하는 방법
  상품분류       가격    상품분류       가격      상품분류_TV/ 상품분류_냉장고 
                                                  상품분류_세탁기/ 상품분류_컴퓨터
    TV           1000000  0            1000000       1            0      0    0    
    냉장고       1500000  1            1500000       0            1      0    0
    세탁기        500000  2             500000       0            0      1    0
    컴퓨터       2000000  3            2000000       0            0      0    1

scaling
ㄴ 데이터 정규화, 표준화

 정규화 : 서로 다른 feature의 크기를
          통일하기 위해서 크기를 변환해 줌
          
 표준화 : 데이터의 feature 각각의 평균이 0이고
          분산이 1인 가우시안 정규분포를
          가진 값으로 변환하기
          
 Standard Scaler
 ㄴ     평균이 0이고, 분산이 1인
        정규분포형태로 변환해 줌
        
 MinMax Scaler
 ㄴ     데이터 값을 0과 1 사이의 값으로
        변환해 줌 : 음수 값은 -1 ~ 1 시이의 값으로
                    변환해 줌
